## Restful API Demo

### Environment:
    - IDEA 2019
    - Maven 3.6.0
    - Spring Boot 2.6.2
    
### Run instruction:
    - need to make sure the db.json path is correct
    - Run DemoController.java to Run Spring Boot
    - open Browser
    - type Following URLs of Test Cases
